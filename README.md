Criação de uma Dashboard em Excel que possa visualizar de forma clara todos os dados necessários já fornecidos da venda do XBOX GAME PASS em 2024. Precisando cumprir esses 4 requisitos:

1 - Visualização das compras totais de TODOS os pacotes, com filtro para tipo de inscrição, podendo ser anuais, trimestrais ou mensais;
2- Visualização do valor total de compra apenas para quem comprou o pacote da EA com filtro para tipo de inscrição, podendo ser anuais, trimestrais ou mensais;
3- Visualização do valor total de compra apenas para quem comprou o pacote do Minecraft com filtro para tipo de inscrição, podendo ser anuais, trimestrais ou mensais;
4- Análise da inscrição em cada mês deste ano, além dos filtros através dos planos Core, Standard e Ultimate juntamente com o filtro do tipo de inscrição, podendo ser mensal, trimestral  e anual;
